# Typora-Theme-Own

## Descriptions about the Themes / 主题介绍

- 各个主题由本人设计制作完成；
- 目前可供使用的主题为==【暮光伯爵：镀金时代】==

## Installation and Usage / 安装与使用

- 将对应的 css 文件及同名文件夹复制到 Typora 的 themes 文件夹下；
- 重启 typora ，即可选择主题进行使用。
